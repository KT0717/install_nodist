## nodist で Node.js をバージョン管理

https://qiita.com/satoyan419/items/56e0b5f35912b9374305

（※）コマンドプロンプトで実行すること

バージョン確認

```
nodist -v
```

インストール可能な Node.js のバージョンの一覧を表示

```
nodist dist
```

ここでは 10.15.0 の Node.js をインストール

```
nodist + 10.15.0
```

使用する Node.js のバージョンを指定

```
nodist 10.15.0
```
